# birthdayMH
As the name tells, this repo is to publish a page for my friend's brithday. I use a NFC sticker to write the URL (a link) to this repo's publish page. And stick the NFC sticker inside the envelop, I also write her a letter. So whenever she scans it with her phone, our image will drop in with wishes.  
To be honest, I wrote this app on the bus going to her birthday, I used AI to write the code and use my iPhone only. What I learned is to create repo, publish pages. And private repo cannot publish pages.  
Oh the image add and delete is when I test it, because the final image is the birthday itself, I did it there and gave it to her. One of my friend was surprised and I actually give it to him so that he have a gift for his girlfriend too, haha.
