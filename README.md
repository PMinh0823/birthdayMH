# birthdayMH

As the name suggests, this repository hosts a birthday webpage for one of my friends.

I wrote the webpage's URL onto an NFC sticker and placed the sticker inside the envelope with a handwritten letter. When she scanned it with her phone, our photos dropped onto the screen along with a birthday message.

To be honest, I built this on the bus on the way to her birthday party. I only had my iPhone, so I used AI to help me write the code. Through this project I learned how to create a GitHub repository, publish a GitHub Pages site, and discovered that private repositories can't be published with GitHub Pages.

The image commits in the history are just from testing. I replaced them with the real birthday photos right before giving her the gift.

One of my friends was so surprised by the idea that I gave him the project too, so he could make a similar gift for his girlfriend. 😄
